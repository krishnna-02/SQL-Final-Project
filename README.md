# 🎓 Student Management System Using SQL

## 📌 Project Overview

The **Student Management System** is a relational database project built using **MySQL** to manage academic information such as students, departments, courses, instructors, and enrollments. The project demonstrates database design, SQL querying, and data management techniques using real-world scenarios.

---

## 🎯 Objectives

- Design a normalized relational database.
- Manage student and course information efficiently.
- Perform CRUD (Create, Read, Update, Delete) operations.
- Practice SQL joins, subqueries, aggregate functions, and window functions.
- Generate meaningful reports using SQL queries.

---

## 🗄️ Database Schema

The project consists of the following tables:

- **Students**
- **Departments**
- **Courses**
- **Instructors**
- **Enrollments**

These tables are connected using **Primary Keys** and **Foreign Keys** to maintain data integrity.

## 📊 Sample SQL Operations

- Add, update, and delete student records.
- Retrieve student enrollment details.
- Join multiple tables to generate reports.
- Calculate average course credits.
- Count student enrollments.
- Generate running totals using window functions.
